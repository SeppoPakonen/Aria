# Phase 07: Logging and Basic Diagnostics

## Goal
Establish a foundational logging system to record application activity, aid in troubleshooting, and provide diagnostic information when errors occur.

## Tasks
1. **01-implement-basic-logging-system**: Set up the `logging` module and configure a log file in `~/.aria/aria.log`.
2. **02-integrate-logging-into-core**: Add log statements to `navigator.py`, `script_manager.py`, and `aria.py`.
3. **03-implement-diagnostics-command**: Add a command (e.g., `aria diag` or `aria --version --verbose`) to show environment information and recent logs.

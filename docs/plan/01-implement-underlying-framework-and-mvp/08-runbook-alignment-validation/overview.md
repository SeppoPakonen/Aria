## Goal
Validate the MVP implementation against the documented runbooks to ensure consistency and correctness. Ensure all described workflows are functional and that the CLI output matches the documentation.

3. **03-validate-browser-commands**: Verify `aria open` and `aria close` in `30-browsers.md`.
4. **04-validate-help-and-diagnostics**: Verify help and version info in `40-settings-help-man.md` and logging in `90-troubleshooting.md`.
5. **05-final-mvp-validation**: Run end-to-end scenarios from `50-recipes.md`.
